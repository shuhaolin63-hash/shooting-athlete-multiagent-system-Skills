> **Спортивные пневматические винтовки в КНР подлежат строгому правовому регулированию. Их комплектация и использование разрешены только утверждённым спортивным стрелковым организациям и коммерческим стрелковым тирам. Частное владение физическими лицами не допускается.**

Пожалуйста, соблюдайте действующие законы и нормативные акты вашей страны/региона. В Китае деятельность, связанная со спортивными пневматическими винтовками, регулируется:
- «Законом КНР об управлении огнестрельным оружием»
- «Уголовным кодексом КНР»

**Условия законного использования (только для следующих организаций):**

- Коммерческие стрелковые тиры, одобренные провинциальными органами общественной безопасности
- Спортивное снаряжение хранится централизованно в специализированных помещениях организации; спортсмены используют его только во время тренировок и соревнований в установленных местах

### Назначение данной системы

**Данная система является исключительно проектом по систематизации знаний и не содержит каких-либо инструкций по эксплуатации, производству, покупке или хранению реального спортивного снаряжения.** Все методы тренировок основаны на теоретических знаниях о стрелковом спорте, физической подготовке, психологической тренировке и академических исследованиях. Тренировки без патронов должны проводиться в законных стрелковых объектах под руководством профессионального тренера.

## Использование (Usage)

Введите слово-триггер в инструменте ИИ-программирования для активации:

```
/shooting-athlete-multiagent
```

### Условия активации

| Категория | Примеры |
|------|------|
| **Прямая активация** | «стрелковая тренировка», «пневматическая винтовка», «винтовка 10 м», «техника стрельбы», «оценка стрельбы» |
| **Косвенная активация** | «вибрация ствола», «точность прицеливания», «контроль спуска», «задержка дыхания при выстреле», «частота сердечных сокращений при стрельбе» |
| **Ситуативная активация** | Составление плана стрелковой тренировки, анализ технических действий, оценка результатов тренировки, формирование команды обеспечения |

### Три пути использования

**Новичок**: Оценка → Формирование без патронов → Физические показатели → Конфигурация команды

**Продвинутый**: Этапная оценка → Тренировка с боеприпасами → Физическая подготовка → Поиск литературы → Анализ данных

**Подготовка к соревнованиям**: Психологическая устойчивость → Подгонка оборудования → Моделирование оценки → Психологическое обеспечение → Предсоревновательная проверка

### Команды управления

| Команда | Описание |
|------|------|
| `/shooting-athlete-multiagent` | Вызов полной системы (оркестратор автоматически направляет запросы) |
| `/shooting-athlete-module1` | Только тренировка формирования без патронов |
| `/shooting-athlete-module2` | Только практическая стрельба |
| `/shooting-athlete-module3` | Только специализированная физическая подготовка |
| `/shooting-athlete-module4` | Только психологическая тренировка под давлением |
| `/shooting-athlete-module5` | Только подгонка и настройка оборудования |

### Обзор 15 агентов

| Категория | Идентификатор агента | Обязанности |
|------|-----------|------|
| **Главный координатор** | orchestrator-agent | Маршрутизация задач, координация агентов, контроль качества |
| **Тренировочный модуль** | module1-aimless-foundation | Формирование без патронов (поза / стабильность / прицеливание / спуск / дыхание) |
| **Тренировочный модуль** | module2-live-fire | Практическая стрельба (стандарт / финальное давление / помехи / командное взаимодействие / разбор) |
| **Тренировочный модуль** | module3-physical-fitness | Специализированная физическая подготовка (кор / плечи-шея / равновесие / кардио / восстановление) |
| **Тренировочный модуль** | module4-mental-resilience | Моделирование соревновательного давления (сопротивление / визуализация / усталость / фиксация ритуалов) |
| **Тренировочный модуль** | module5-equipment-fitting | Подгонка оружия и экипировки (настройка оружия / корректировка костюма / адаптация к новым правилам) |
| **Общий** | literature-support-agent | Поиск и авторитетное цитирование 20 научных статей PubMed |
| **Общий** | scale-eval-agent | Пятимерная количественная оценка и отслеживание прогресса |
| **Общий** | data-calc-agent | Расчёт физических показателей, анализ тренировочных данных |
| **Командная роль** | team-coaching | Тренерская группа (главный тренер + помощники + первый тренер) |
| **Командная роль** | team-strength | Команда научно-исследовательской физической подготовки (главный тренер + помощник + наука) |
| **Командная роль** | team-psychology | Психологическое обеспечение спорта (штатный психолог) |
| **Командная роль** | team-medical | Медицинская реабилитационная группа (врач + 2 реабилитолога) |

| Роль | Сборная A (8-11 чел.) | Самоокупаемость за рубежом B1 (5-6 чел.) | Ассоциативное совместное B2 (минимум) |

## В заключение

> Стрельба — это спорт на 10% техники и 90% психологии.
>
> Но 90% психологии требуют 100% техники для своей реализации.
>
> Без формирования навыка с незаряженным оружием до уровня мышечной памяти, никакая психологическая стабильность не даст вам «десятки».
> Без физической подготовки, позволяющей удерживать оружие без дрожи в течение часа, в последнем выстреле предаст вас ваша рука.
> Без подгонки оборудования к вашему телу, вы соревнуетесь не только с соперником, но и со своей собственной винтовкой.
>
> Эта система деконструирует всё перечисленное в 15 агентов, каждый из которых сосредоточен на одной задаче — и выполняет её на пределе.
> Она не нажмёт спуск за вас — но она расскажет, почему ваш спуск каждый раз уходит на 0,3 мм в сторону,
> и что стоят за этими 0,3 мм — проблема техники, проблема физики, проблема психологии или проблема оборудования.
>
> А затем предложит структурированное решение.
>
> Не куриный бульон. Баллистика.
