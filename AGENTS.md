# 全局智能体总注册表

> 10米气步枪射击训练多智能体系统 — 15个智能体注册、协作与技能管理

---

## 系统架构图

```
                              ┌──────────────────────┐
                              │  orchestrator-agent  │
                              │     总调度智能体       │
                              └──────────┬───────────┘
                                         │
                    ┌────────────────────┼────────────────────┐
                    │                    │                     │
         ┌──────────┴──────────┐  ┌──────┴──────┐  ┌──────────┴──────────┐
         │   5个训练模块智能体   │  │ 3个通用智能体 │  │  6个团队角色智能体    │
         │                     │  │             │  │                     │
         │ module1 无弹定型    │  │ literature  │  │ team-coaching 教练组  │
         │ module2 实弹射击    │  │ scale-eval  │  │ team-strength 体能   │
         │ module3 体能实操    │  │ data-calc   │  │ team-psychology 心理  │
         │ module4 心理抗压    │  │             │  │ team-medical 医疗    │
         │ module5 装备适配    │  │             │  │ team-armorer 枪械    │
         └─────────────────────┘  └─────────────┘  │ team-logistics 后勤   │
                                                    └─────────────────────┘
```

**设计原则**：
- 每个智能体专注单一职责域，避免功能重叠
- 所有智能体通过 orchestrator-agent 协调，不直接互相调用
- 文献支撑智能体为所有其他智能体提供学术证据支持
- 数据计算智能体为所有量化需求提供计算服务
- 训练模块智能体按五大训练模块拆分，每个模块独立负责一个训练领域
- 团队角色智能体模拟真实保障团队的六个核心角色，覆盖三种体制差异

## 1. orchestrator-agent（总调度智能体）

### 基本信息

| 字段 | 内容 |
|------|------|
| **名称** | 总调度智能体 |
| **标识符** | `orchestrator-agent` |
| **目录** | `agents/orchestrator-agent/` |
| **优先级** | 最高（系统入口） |

### 职责描述

总调度智能体是多智能体系统的统一入口和中枢控制器，承担以下核心职责：

1. **任务路由**：解析用户输入，识别意图类别，将任务精准分派至对应专业智能体
2. **智能体协调**：管理多智能体协作流程，处理跨智能体的复合任务
3. **输出质量控制**：审核各智能体输出的准确性、一致性和完整性
4. **上下文管理**：维护对话上下文，确保多轮交互的一致性
5. **使用路径引导**：根据用户水平（新手/进阶/比赛）推荐最优智能体调用序列

### 输入规格

| 输入类型 | 描述 | 示例 |
|----------|------|------|
| 用户自然语言查询 | 射击训练相关的任意问题 | "如何提高握持稳定性？" |
| 训练数据 | 射击训练记录数据 | 弹着点数据、SCATT数据 |
| 评估请求 | 评分或评估需求 | "帮我评估当前训练水平" |
| 文献检索请求 | 学术论文查询 | "关于射击心率的研究有哪些？" |

### 输出规格

| 输出类型 | 描述 |
|----------|------|
| 任务分派指令 | 指向具体智能体的调用指令及参数 |
| 整合报告 | 多智能体输出的汇总分析报告 |
| 路径推荐 | 基于用户水平的使用路径建议 |
| 质量审核反馈 | 对下游智能体输出的修正或补充 |

### 协作关系

```
orchestrator-agent
    ├── 委派训练任务 → module1-aimless-foundation
    ├── 委派训练任务 → module2-live-fire
    ├── 委派训练任务 → module3-physical-fitness
    ├── 委派训练任务 → module4-mental-resilience
    ├── 委派训练任务 → module5-equipment-fitting
    ├── 委派文献查询 → literature-support-agent
    ├── 委派评估任务 → scale-eval-agent
    ├── 委派计算任务 → data-calc-agent
    ├── 委派团队问题 → team-coaching / team-strength / team-psychology
    │                    team-medical / team-armorer / team-logistics
    └── 汇总所有智能体输出 → 生成综合报告
```

### 可加载技能

| 技能名称 | 描述 | 路径 |
|----------|------|------|
| 意图识别 | 识别用户查询意图类别 | `skills/shared-common/` |
| 路径推荐 | 根据用户水平推荐使用路径 | `skills/shared-common/` |
| 输出审核 | 质量控制与一致性检查 | `skills/shared-common/` |

## 2. module1-aimless-foundation（模块1：无弹基础定型智能体）

### 基本信息

| 字段 | 内容 |
|------|------|
| **名称** | 无弹基础定型智能体 |
| **标识符** | `module1-aimless-foundation` |
| **目录** | `agents/module1-aimless-foundation/` |
| **优先级** | 高（基础模块） |

### 职责描述

负责射击运动员的空枪基础定型训练，是一切后续实弹训练的根基：

1. **持枪姿态定型**：立姿/跪姿/卧姿三姿标准化框架建立
2. **手部微稳定专项**：叠弹壳训练、控抖训练、手指独立性练习
3. **瞄准标准化**：准星平正、焦点锁定、光线抗干扰训练
4. **击发规范**：空扣扳机练习、屏息击发、跟随动作标准化
5. **呼吸控制**：腹式呼吸、屏息节奏、高心率条件下呼吸调节

### 场景区分

| 场景 | 侧重内容 |
|------|----------|
| 室内 | 光照可控，聚焦抗干扰训练；精细动作重复定型 |
| 室外 | 强光/逆光/风力影响下持枪稳定性训练 |

### 替代方案

| 条件 | 替代方案 |
|------|----------|
| 居家 | 空枪对镜练习、瓶装水负重训练、墙壁贴纸靶+激光笔瞄准 |
| 室内（无靶场） | 激光训练器、配重模拟枪、SCATT空枪训练系统 |

### 学术支撑

- Ihalainen 2016 — 射击姿态稳定性研究
- Era 1996 — 手部微控制与年龄相关变化
- Park 2019 — 瞄准精度与视觉焦点关系

### 输入规格

| 输入类型 | 描述 | 示例 |
|----------|------|------|
| 当前水平 | 用户训练阶段描述 | "新手，持枪3个月" |
| 场景条件 | 室内/室外/居家/靶场 | "只有居家条件" |
| 身体数据 | 身高、体重、臂长等 | 通过 data-calc-agent 获取 |
| 问题反馈 | 定型过程中的具体问题 | "立姿重心不稳" |

### 输出规格

| 输出类型 | 描述 |
|----------|------|
| 定型训练计划 | 分阶段空枪定型方案（周/日计划） |
| 动作要领详解 | 三姿标准化动作分解与要点说明 |
| 替代方案 | 场景受限时的等效训练方案 |
| 常见错误纠正 | 典型问题诊断与修正方法 |
| 呼吸节奏方案 | 个性化屏息/呼吸训练方案 |

### 协作关系

```
module1-aimless-foundation
    ├── 请求文献支撑 → literature-support-agent
    ├── 请求数据计算 → data-calc-agent
    ├── 请求评估打分 → scale-eval-agent
    └── 接收 orchestrator-agent 的任务委派
```

### 可加载技能

| 技能名称 | 描述 | 路径 |
|----------|------|------|
| 无弹定型训练 | 空枪练习标准化流程 | `skills/shooting-training-scale/` |
| 持枪姿态标准化 | 三姿框架建立与检查 | `skills/shooting-training-scale/` |
| 手部微稳定专项 | 叠弹壳/控抖训练体系 | `skills/shooting-training-scale/` |
| 呼吸控制训练 | 腹式呼吸与屏息节奏 | `skills/shooting-training-scale/` |

## 3. module2-live-fire（模块2：实弹射击实操智能体）

### 基本信息

| 字段 | 内容 |
|------|------|
| **名称** | 实弹射击实操智能体 |
| **标识符** | `module2-live-fire` |
| **目录** | `agents/module2-live-fire/` |
| **优先级** | 高（核心模块） |

### 职责描述

负责从基础实弹到高压赛事的完整实弹射击训练体系：

1. **基础常规轮次实弹**：60发/10发一组逐发记录、弹着群分析、节奏建立
2. **决赛高压专项**：奥运淘汰制模拟、逐发计分训练、双人团体联动、落后失误场景应对
3. **抗干扰实弹**：噪音闪光干扰、观众音效模拟、盲打训练、异地温差拉练
4. **双人团体协同**：混合团体同步呼吸击发、分数互相施压、团队默契培养
5. **单发精细化复盘**：高速录像逐帧分析、靶纸逐发拆解、激光弹道量化分析

### 场景区分

| 场景 | 侧重内容 |
|------|----------|
| 室内 | 比赛流程高压模拟；精准弹着群分析 |
| 室外 | 风/温度/光线变化干扰实弹；环境适应性训练 |

### 替代方案

| 条件 | 替代方案 |
|------|----------|
| 无实弹条件 | SCATT/MegaLink空枪训练系统模拟实弹流程 |
| 无靶场 | 激光射击训练器、电子靶投影系统 |

### 学术支撑

- Lang & Zhou 2024 — 射击表现与心理因素关系
- Ihalainen 2016b — 实弹训练效果研究
- Mullineaux 2012 — 射击技术动作分析
- Mononen 2003 — 射击表现预测模型
- Ball 2003 — 射击精度影响因素

### 输入规格

| 输入类型 | 描述 | 示例 |
|----------|------|------|
| 训练类型 | 常规/决赛/抗干扰/团体 | "决赛高压训练方案" |
| 当前成绩 | 近期实弹成绩数据 | "60发580环" |
| 弹着点数据 | 靶纸弹着分布 | 通过 data-calc-agent 分析 |
| 团体赛需求 | 混合团体训练需求 | "混团同步击发训练" |

### 输出规格

| 输出类型 | 描述 |
|----------|------|
| 实弹训练计划 | 分层渐进式实弹方案 |
| 单发复盘报告 | 逐发技术动作分析与改进建议 |
| 弹着群分析 | 分布参数、散布模式诊断 |
| 决赛模拟方案 | 高压淘汰制训练流程 |
| 团体协同方案 | 混合团体配合训练计划 |

## 4. module3-physical-fitness（模块3：射击专项体能实操智能体）

### 基本信息

| 字段 | 内容 |
|------|------|
| **名称** | 射击专项体能实操智能体 |
| **标识符** | `module3-physical-fitness` |
| **目录** | `agents/module3-physical-fitness/` |
| **优先级** | 高（体能模块） |

### 职责描述

负责射击运动员的专项体能训练，核心目标是支撑长时间持枪稳定性：

1. **躯干核心静力**：平板支撑（多角度变体）、侧支撑、核心抗旋转训练
2. **肩颈手臂静力负重托举**：肩部静力保持、颈深屈肌训练、前臂握力耐力
3. **平衡前庭感知训练**：单脚站立变体、BOSU球训练、前庭刺激适应
4. **有氧心肺训练**：降低静息心率、提升有氧基础、耐力基础建设
5. **赛后筋膜松解劳损修复**：泡沫轴松解、筋膜球按压、拉伸恢复方案

### 场景区分

| 场景 | 侧重内容 |
|------|----------|
| 室内 | 健身房器械辅助训练；精准负荷控制 |
| 室外 | 跑步+自然环境平衡训练；综合体能拉练 |

### 替代方案

| 条件 | 替代方案 |
|------|----------|
| 居家（无器械） | 自重训练（平板支撑变体/墙壁推手/单脚站立） |
| 简易器械 | 弹力带（肩部静力/核心抗旋转）、居家泡沫轴（筋膜松解） |

### 学术支撑

- Zemková 2022 — 射击运动员核心稳定性研究
- Lee & Shin 2025 — 射击专项体能训练效果
- Hrysomallis 2011 — 平衡训练与运动表现
- Sadowska 2019 — 射击运动员体能评估

| 场景条件 | 健身房/居家/室外 | "只有居家条件" |
| 训练阶段 | 赛季/休赛期 | "备战期" |

### 输出规格

## 5. module4-mental-resilience（模块4：赛场高压心理模拟实操智能体）

### 基本信息

| 字段 | 内容 |
|------|------|
| **名称** | 赛场高压心理模拟实操智能体 |
| **标识符** | `module4-mental-resilience` |
| **目录** | `agents/module4-mental-resilience/` |
| **优先级** | 高（心理模块） |

### 职责描述

负责射击运动员的心理韧性训练，核心目标是高压环境下稳定发挥：

1. **队内比分对抗/高压逆境模拟**：模拟落后追分、关键最后一发、淘汰边缘压力场景
2. **赛事沉浸式可视化冥想**：比赛场景心理预演、成功经验锚定、流程化心理剧本
3. **极限疲劳射击特训**：高强度训练后进行射击测试，训练疲劳状态下专注力
4. **每日心态记录复盘**：情绪日志、压力源识别、心态波动模式分析
5. **个人稳定仪式固化**：赛前准备仪式、发枪间流程、击发前微习惯建立

| 场景 | 侧重内容 |

| 室内 | 可控高压模拟；标准化心理训练环境 |

| SCATT原始数据 | 训练系统导出数据 | 晃动曲线、击发参数 |
| 时间序列数据 | 多次训练的成绩序列 | 最近12周成绩记录 |

### 输出规格

| 输出类型 | 描述 |
|----------|------|

| 弹着群分析报告 | 分布参数、趋势图 |
| 技术参数评分 | 各技术维度量化评分 |
| 训练负荷分析 | 训练量与恢复状态评估 |
| 趋势预测 | 基于历史数据的进步趋势 |

    ├── 请求文献支撑 → literature-support-agent（获取计算公式学术依据）
