# 全Agent共享通用技能

> 为射击训练多智能体系统提供底层共享能力

## 功能模块

### 1. 文本检索
- 关键词提取与匹配
- 多语言文本检索（中/英）
- 上下文相关性排序

### 2. 表格生成
- Markdown表格生成
- 结构化数据展示
- 对比分析表格

### 3. 数据摘要
- 训练数据汇总统计
- 进步率计算
- 异常值检测与标注

### 4. 引用格式化
- APA格式引用生成
- PubMed标准引用
- 中文引用格式
- DOI/PMID链接嵌入

### 5. 通用工具
- 单位换算（mm/cm/m、秒/分钟、bpm等）
- 百分比/比率计算
- 简单统计分析（均值、标准差、趋势）
