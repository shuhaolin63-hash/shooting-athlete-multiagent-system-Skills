## 二、专业射击皮衣标注图

### 2.1 概述

专业射击皮衣（射击服/皮衣）是ISSF规则允许使用的标准装备，其主要功能是限制关节活动范围，提供额外的身体稳定性。ISSF规则对皮衣的厚度、硬度、层数等均有严格限制。

### 2.2 标注内容

| 标注区域 | 说明 | ISSF规则限制 |
|---------|------|-------------|
| **厚度分布** | 皮衣在不同部位的厚度不同，关节处较薄（允许弯曲），躯干处较厚（提供支撑） | 最大厚度：躯干不超过2.5层（含内衬），关节不超过1层 |
| **调节扣位置** | 皮衣上设有多个调节带和扣环，用于调整松紧度 | 允许使用调节装置，但不得增加额外刚性支撑 |
| **配重块安装位** | 皮衣上预设有配重块挂载点，可安装铅块等配重以调整重心 | 总配重不超过ISSF规则限值 |
| **肩部加固区** | 肩部使用多层硬质材料加固，减少肩部晃动 | 肩部材料硬度不超过规定值 |
| **背部支撑条** | 背部纵向支撑条，辅助脊柱保持稳定 | 不得使用金属或超硬材料 |
| **袖口约束** | 袖口设计为约束手臂活动范围 | 袖长和松紧度需符合规则 |

### 2.3 皮衣调试要点

```
调试顺序（从前到后）：

  1. 领口 → 确保贴合适中，不影响转头动作
  2. 肩部 → 左肩下沉后皮衣肩部贴合但不压迫
  3. 胸前 → 拉紧胸前调节带，固定躯干
  4. 腰部 → 收紧腰带，提供核心支撑
  5. 背部 → 检查背部支撑条贴合脊柱
  6. 袖口 → 调整袖口松紧，确保手臂活动受限但不过紧
  7. 配重 → 挂载配重块，调整前后左右重心平衡
```

## 三、气步枪枪托调节标注图

### 3.1 枪托可调部件一览

| 部件名称 | 调节范围 | 调节精度 | 作用 |
|---------|---------|---------|------|
| **贴腮板高度** | 上下调节约30-50mm | 1-2mm/格 | 确保右眼与瞄准镜光轴对齐 |
| **贴腮板前后** | 前后调节约20mm | 1-2mm/格 | 配合面部前后位置微调 |
| **托肩板角度** | 角度调节约15-30度 | 1度/格 | 适配不同肩部角度和抵肩位置 |
| **握把角度** | 左右旋转约30度 | 2-3度/格 | 适配右手握持舒适度和扳机手指位置 |
| **握把前后** | 前后调节约15mm | 2mm/格 | 配合手臂长度微调 |
| **枪托长度** | 前后调节约30-50mm | 2mm/格 | 适配运动员臂展 |
| **枪托倾斜角** | 俯仰调节约5度 | 0.5度/格 | 微调枪口自然指向高度 |

### 3.2 调节标注图说明

```
枪托标注图应清晰标注以下要素：

  [侧面视图]
  ┌──────────────────────────────────┐
  │                                  │
  │  ① 贴腮板 ← 可上下/前后调节      │  ← 高度调节螺丝
  │     │                            │  ← 前后调节螺丝
  │     ├─ 枪托主体                  │
  │     │                            │
  │  ② 托肩板 ← 角度可调            │  ← 角度调节旋钮
  │     │                            │
  │  ③ 握把 ← 角度/前后可调         │  ← 旋转锁紧螺丝
  │     │                            │  ← 前后调节螺丝
  │     │                            │
  │  ④ 枪托长度调节 ← 底部滑轨      │  ← 长度调节螺丝
  │     │                            │
  │  ⑤ 枪托俯仰角 ← 可微调          │  ← 俯仰调节螺丝
  │                                  │
  └──────────────────────────────────┘

  [标注颜色编码]
  红色 → 调节螺丝/旋钮位置
  蓝色 → 可调范围标注
  绿色 → 推荐初始位置参考
  黄色 → ISSF规则限值标注
```

## 四、拳托（手掌托）标注图

### 4.1 概述

拳托（Palm Rest / Hand Stop）安装在枪身前部，用于左手掌心托住枪身，是立姿射击中左臂与枪身之间的重要连接件。

### 4.2 标注内容

| 标注项 | 说明 | 调节要点 |
|--------|------|---------|
| 安装位置 | 枪身前部导轨上，位置可前后滑动 | 前后位置影响左臂角度和重心分布 |
| 托面角度 | 托面与枪身的夹角可调 | 适配左手掌心的自然角度 |
| 托面材质 | 通常为铝合金或碳纤维，表面有防滑纹理 | 不得使用超硬材料（ISSF规则限制） |
| 高度 | 部分型号支持高度微调 | 影响枪身整体水平度 |
| ISSF限制 | 拳托不得向前超出枪管前端面 | 比赛前需通过装备检查 |

### 4.3 使用要点

- 左手掌心平放在拳托上，手指自然放松不握枪
- 拳托位置调整到肘部恰好锁定在髋骨支撑点
- 拳托角度调整到手腕保持自然中立位（不弯曲）

## 五、扳机组调节标注图

### 5.1 扳机组结构

| 部件 | 功能 | 可调参数 |
|------|------|---------|
| 扳机 | 手指直接接触的部件 | 行程长度、压力值 |
| 扳机第一行程 | 从手指接触扳机到扳机止点的距离 | 通常5-15mm可调 |
| 扳机第二行程 | 从止点到击发点的距离 | 通常0.5-3mm可调 |
| 扳机压力 | 扣动扳机所需的最小力 | ISSF最低要求500g |
| 扳机返回弹簧 | 击发后扳机自动回位的力量 | 影响扳机回位速度 |
| 扳机横向位置 | 扳机相对握把的左右位置 | 适配食指长度和角度 |

### 5.2 ISSF扳机规则

| 规则项 | 限制值 | 备注 |
|--------|--------|------|
| 最小扳机压力 | 500g | 气步枪项目最低扳机压力 |
| 允许误差 | +/-10g | 即最低不小于450g |
| 调试工具 | 专用扳机测力计 | 枪械师使用精密测力计校准 |
| 赛前检查 | ISSF装备检查站随机抽检 | 不合格将被取消比赛资格 |

### 5.3 扳机调节标注图说明

```
扳机组侧面标注图：

  ┌──────────────────────────┐
  │  扳机                     │ ← 食指接触面
  │   │                       │
  │   ├─ 第一行程（5-15mm）    │ ← 可调
  │   │                       │
  │   ├─ 止点（Stop Point）    │ ← 可调位置
  │   │                       │
  │   ├─ 第二行程（0.5-3mm）   │ ← 可调
  │   │                       │
  │   ├─ 击发点（Release）     │ ← 扳机释放，击锤释放
  │   │                       │
  │   └─ 返回弹簧              │ ← 回位速度可调
  │                           │
  │  调节螺丝位置：             │
  │  ① 第一行程长度螺丝        │ ← 扳机后方
  │  ② 止点位置螺丝            │ ← 扳机上方
  │  ③ 第二行程长度螺丝        │ ← 扳机前方
  │ ④ 扳机压力弹簧螺丝         │ ← 扳机组内部
  │ ⑤ 扳机横向位置螺丝         │ ← 扳机组侧面
  └──────────────────────────┘

  关键数值：
  - 第一行程：推荐8-12mm（太短容易误触，太长影响精度）
  - 第二行程：推荐1-2mm（需有清晰的手感反馈）
  - 扳机压力：推荐500-600g（ISSF最低500g，留适当余量）
```

## 六、训练器材说明

### 6.1 叠弹壳测试

| 项目 | 说明 |
|------|------|
| **器材** | 5.56mm（.223）空弹壳1枚 |
| **方法** | 将弹壳竖直叠放在枪管口（消音器前端），据枪后弹壳保持不倒 |
| **判定** | 弹壳稳定保持10秒以上为合格，30秒以上为优秀 |
| **目的** | 测试据枪稳定性，弹壳倒下说明枪口晃动超过弹壳底面直径 |
| **难度** | 初学者几乎无法完成，国家队选手可稳定保持30-60秒 |
| **注意** | 仅限空枪训练，严禁实弹状态下进行 |

### 6.2 SCATT激光弹道测试仪

| 项目 | 说明 |
|------|------|
| **全称** | SCATT（Shooting Competition Analysis Training Tool） |
| **功能** | 实时追踪枪口瞄准轨迹，记录完整的据枪-瞄准-击发过程 |
| **输出数据** | 瞄准轨迹图、晃动幅度、击发瞬间偏移、据枪时长等 |
| **使用方式** | 安装在枪管前端，对准训练用感应靶（无需实弹） |
| **训练价值** | 数据化分析个人晃动模式，精准识别技术问题 |
| **型号** | SCATT MX-02（最新款）、SCATT USB（入门款） |
| **价格参考** | 约5-10万元/套 |

### 6.3 足底压力传感器

| 项目 | 说明 |
|------|------|
| **功能** | 实时检测站立时足底压力分布 |
| **应用** | 检查立姿射击时双脚重心分配是否合理（目标：前脚掌70%） |
| **使用方式** | 传感器垫铺设在射击位置地面，运动员站在上面据枪 |
| **输出数据** | 前后重心比、左右重心比、压力中心轨迹、重心稳定性 |
| **训练价值** | 及时发现重心偏移问题，指导站姿调整 |
