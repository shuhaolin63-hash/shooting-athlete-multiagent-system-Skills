# 选手训练参考图集说明

> 本文件夹用于存放国内顶尖射击选手的训练实拍参考图片，供教练员和运动员进行技术对比学习。

---

## 一、文件夹用途

- **目的**：收集整理国内顶尖射击选手的标准训练动作参考图
- **用途**：技术动作对比分析、姿势标准化参考、教学示范材料
- **使用者**：主教练、助理教练、运动员
- **使用方式**：与个人训练视频/照片进行对照，找出技术偏差并纠正

## 四、图片收集标准

### 技术参考照片要求

1. **清晰度**：分辨率不低于1920x1080，关键部位（肩、肘、手、脚）清晰可见
2. **角度**：优先收集侧面90度视角（展示完整的立姿据枪轮廓）
3. **完整度**：全身入镜，包含从脚到枪口的完整姿势
4. **时机**：据枪稳定后、击发前的静止状态最佳
5. **装备**：穿着标准射击皮衣、使用标准比赛装备的状态

### 体能训练照片要求

1. **场景**：在射击训练馆或专项体能训练区域
2. **器材**：使用射击专项体能训练器材（如配重块、平衡板、SCATT系统等）
3. **动作**：展示完整训练动作，非摆拍状态
4. **全身**：全身入镜，展示训练姿势和器材使用方式
