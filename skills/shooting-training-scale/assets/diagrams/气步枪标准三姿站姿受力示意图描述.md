## 二、身体各部位详细标注

### 1. 双脚站位

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 脚间距 | 与肩同宽（约35-45cm） | 提供稳定的基础支撑面 |
| 左脚朝向 | 微向前15度 | 左脚（支撑脚）略微前旋，增强前后稳定性 |
| 右脚朝向 | 微向外15-30度 | 右脚（平衡脚）外旋辅助重心分配 |
| 重心分布 | 70%前脚掌，30%后脚跟 | 前脚掌承重为主，脚趾抓地增强稳定性 |
| 膝关节 | 微屈5-10度，不锁死 | 微屈提供缓冲，锁死易疲劳 |

**关键要点**：双脚站位的稳定性是整个立姿的基础，重心偏移1cm可导致枪口偏差数毫米。

---

### 2. 骨盆-脊柱

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 骨盆倾斜 | 微后倾 | 骨盆后倾使重心前移，降低躯干不稳定性 |
| 脊柱曲线 | 保持自然S曲线 | 不刻意挺直或弯曲，维持脊柱自然生理弯曲 |
| 躯干前倾角 | 5-8度 | 微前倾使躯干重心与枪械重心趋近对齐 |
| 腰部 | 保持自然 | 不塌腰、不反弓 |

**关键要点**：脊柱的自然曲线是力量传导的核心通道，任何人为改变都会降低稳定性。

---

### 3. 肩带

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 左肩（支撑肩） | 自然下沉 | 下沉的左肩提供更稳固的托枪平台 |
| 右肩（抵肩侧） | 放松贴身 | 右肩不耸起，自然贴合枪托抵肩板 |
| 肩胛骨 | 自然回缩 | 双肩胛骨微微向脊柱方向回缩，稳定肩带 |
| 肩部紧张度 | 左紧右松 | 左肩承重需稳固，右肩需放松避免紧张传导 |

**关键要点**：肩带紧张不对称是立姿的核心特征，左肩稳定托枪、右肩放松传导是标准要求。

---

### 4. 左臂支撑（主力支撑臂）

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 上臂与躯干夹角 | 60-70度 | 最佳支撑角度，兼顾稳定性和舒适性 |
| 肘部支撑点 | 髋骨或肋骨上缘 | 肘部锁定在骨性支撑点，避免肌肉持续发力 |
| 左手位置 | 托住枪身前部/掌托 | 左手掌心向上托枪，或使用专用拳托 |
| 前臂角度 | 前臂尽量垂直地面 | 减少肌肉参与，利用骨骼支撑 |
| 肘关节 | 锁定不可弯曲 | 肘部锁定后形成刚性三角支撑结构 |

**关键要点**：肘部支撑点的选择是立姿稳定性的核心——必须落在髋骨或肋骨的硬支撑面上，而非悬空。

---

### 5. 右手击发（扣扳机手）

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 食指 | 单独扣扳机 | 仅食指第一关节参与扣扳机，其余手指不施力 |
| 余三指 | 轻握枪托 | 中指、无名指、小指轻轻贴合枪托握把，不紧握 |
| 拇指 | 自然放置 | 拇指搭在枪托侧面或握把上方，不施力 |
| 手腕 | 保持平直 | 手腕不弯曲，避免紧张传导到扳机 |
| 扣扳机力度 | 均匀渐进 | 采用渐进式扣压，而非突然扣动 |

**关键要点**：扳机控制的核心是"食指独立运动"，任何多余的手指施力都会导致枪口偏移。

---

### 6. 头部

| 参数 | 标准值 | 说明 |
|------|--------|------|
| 瞄准眼 | 右眼贴瞄准镜 | 右眼紧贴瞄准镜目镜，左眼自然闭合或半闭 |
| 面部肌肉 | 完全放松 | 面部、咬肌、眼周肌肉全部放松，避免紧张传导 |
| 牙齿 | 微张 | 上下牙齿微张1-2mm，避免咬合紧张影响稳定性 |
| 头部角度 | 自然前倾15-20度 | 不过度低头或仰头，保持颈椎自然角度 |
| 贴腮力度 | 适当贴合 | 贴腮板贴合面部即可，不紧压 |

**关键要点**：头部是"最后一环传导"，面部任何肌肉紧张都会通过骨骼传导到枪口。

## 三、力的传递链

```
力的传递路径（自下而上）：

 地面反作用力
      │
      ▼
   双脚（前脚掌70%承重）
      │
      ▼
   踝关节 → 小腿 → 膝关节
      │
      ▼
   大腿 → 髋关节
      │
      ▼
   骨盆（微后倾稳定重心）
      │
      ▼
   脊柱（自然S曲线传导）
      │
      ▼
   肩带（左肩下沉托枪，右肩放松抵枪）
      │
   ┌──┴──┐
   ▼      ▼
左臂支撑  右手击发
（骨骼刚性）（食指独立）
   │      │
   ▼      ▼
枪身前部  扳机组
   │      │
   └──┬───┘
      ▼
   枪管 → 瞄准镜 → 靶心
```

**传递效率要求**：
- 地面到枪身的力传递应尽可能刚性（减少弹性环节）
- 肌肉参与应降到最低（骨骼支撑为主）
- 右侧（击发侧）应完全放松（仅食指工作）

## 五、常见错误标注

| 错误类型 | 具体表现 | 后果 | 纠正方法 |
|---------|---------|------|---------|
| 后仰 | 躯干后仰超过10度 | 重心后移，枪口上跳，弹着点偏高 | 骨盆微后倾，身体微前倾 |
| 耸肩 | 双肩上提，尤其左肩 | 肩部肌肉紧张，疲劳加剧，晃动增大 | 左肩自然下沉，肩胛骨回缩 |
| 肘部悬空 | 左肘未落在髋骨支撑点 | 无骨性支撑，全靠肌肉维持，极不稳定 | 调整肘部位置，确保落在髋骨/肋骨硬支撑 |
| 重心后移 | 重心偏向脚跟（>50%） | 前后稳定性下降，枪口晃动增大 | 前脚掌承重70%，可通过足底压力传感器验证 |
| 咬牙 | 上下牙紧咬 | 咬肌紧张传导至头部，影响瞄准稳定性 | 牙齿微张1-2mm |
| 紧握枪托 | 右手四指紧握 | 肌肉紧张传导扳机，导致急扣或方向偏差 | 仅食指扣扳机，余指轻放 |
| 头歪斜 | 头部侧倾或过度低头 | 颈部肌肉紧张，瞄准线不一致 | 保持头部中正，颈椎自然角度 |

## 六、示意图设计建议

1. **整体构图**：侧面视角，射手面向右侧，靶纸在右远处
2. **人体轮廓**：使用简洁的线条勾勒人体轮廓，不画面部细节
3. **力的传递线**：用粗红色箭头从脚底到枪口标出力传递路径
4. **角度标注**：在关键关节处用蓝色弧线标注角度值
5. **错误标注**：在示意图旁用红色叉号标出常见错误姿势
6. **支撑点高亮**：用黄色高亮标注肘部髋骨支撑点和前脚掌重心区
