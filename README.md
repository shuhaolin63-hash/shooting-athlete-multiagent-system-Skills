## ⚠️ 合规声明（Compliance Statement）

### 中国境内运动用气步枪相关规定

> **运动用气步枪在中国境内受法律法规严格管控，仅限经批准的射击运动单位和营业性射击场合法配置使用，个人不得私自持有。**

请遵守所在国家/地区的相关法律法规。在中国境内，运动用气步枪的相关活动受以下法律法规约束：
- 《中华人民共和国枪支管理法》
- 《中华人民共和国刑法》
- 相关部门制定的实施细则与技术标准

**合法使用条件（仅限以下单位）**：
- 经省级体育行政主管部门批准的射击竞技体育运动单位
- 经省级公安机关批准的营业性射击场
- 运动器材由单位统一保管于专用设施中，运动员仅在训练和比赛期间在指定场所使用

### 本系统定位

**本系统是一项纯知识整理项目，不涉及任何实物运动器材的操作指导、制造、买卖或持有建议。** 所有训练方法均围绕射击运动的理论知识、体能训练、心理训练和学术研究展开。无弹定型训练（空枪练习）应在合法射击训练场所、在专业教练指导下进行。

**比赛准备**：心理抗压 → 装备适配 → 模拟评分 → 心理保障 → 赛前检查

## 写在最后

> 射击是10%技术、90%心理的运动。
>
> 但90%的心理需要100%的技术去承载。
>
> 没有空枪定型到肌肉记忆的程度，心理再稳定也打不出10环。
> 没有核心静力托举到一小时不抖的体能，最后一发的手会出卖你。
> 没有装备与身体合一的适配，你对抗的不只是对手，还有你自己的枪。
>
> 这个系统把这一切解构为15个智能体，每个智能体专注一件事，做到极致。
> 它不会替你扣扳机——但它会告诉你，为什么你的扳机总是扣歪了0.3毫米，
> 以及那0.3毫米对应的，到底是技术问题、体能问题、心理问题，还是装备问题。
>
> 然后给你一个结构化的解决方案。
>
> 不是鸡汤，是弹道学。
